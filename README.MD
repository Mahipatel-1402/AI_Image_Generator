📌 AI Image Generator

📖 Project Description

AI Image Generator is a web-based application that generates images from text prompts using Artificial Intelligence. Users can enter a description of the image they want, select parameters such as model type, number of images, and aspect ratio, and the system will generate AI-created images automatically. 🎨🤖

The project provides a clean and interactive user interface where users can easily create images based on their imagination. It uses an external AI model API to convert text prompts into visual images.

This project demonstrates the integration of frontend web technologies with AI APIs to build an intelligent and creative application.

✨ Features

🧠 Generate images using AI text prompts

🎨 Multiple image generation models

🖼️ Generate multiple images at once

📐 Choose different aspect ratios

🌙 Dark / Light theme toggle

⚡ Fast API-based image generation

🖥️ Simple and responsive UI

🛠️ Technologies:

🌐 HTML5

🎨 CSS3

⚡ JavaScript

🤗 Hugging Face API
